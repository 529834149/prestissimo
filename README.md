prestissimo (composer plugin)
=================================

A [composer](https://getcomposer.org) parallel install plugin.

## Depends

- PHP >=5.3, (suggest >=5.5)
- ext-curl

## Install

```bash
$ composer global require hirak/prestissimo
```

## Uninstall

```bash
$ composer global remove hirak/prestissimo
```
